# 🛒 Online Store Project

## Overview
This is a full-featured **Online Store System** built with **PHP**, **MySQL**, and modern web design.  
The project allows customers to browse products, manage orders, and update personal information via a clean and responsive interface.

The design uses **Glassmorphism** for a sleek, modern look with **emerald and slate color accents**.

---

## Features
- **Customer Dashboard**: View orders, add products, and manage personal info.
- **Order Management**: Track, add, and update orders easily.
- **Responsive & Modern UI**: Glassmorphism design, grid layout, smooth hover effects.
- **Secure Authentication**: Login system with `admin` and `customer` roles.
- **Database Driven**: Fully functional with MySQL (DDL & DML scripts included).
- **User-friendly**: Clean navigation with buttons, emoji cues, and interactive panels.

---

## Technologies
- **Frontend**: HTML, CSS (Glassmorphism, Grid), JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Version Control**: Git & GitHub
